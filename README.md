
# React Eğitim Web Sayfası

React tabanlı eğitim web sayfası projesi. Bu proje, Vite, TypeScript ve Tailwind CSS kullanılarak oluşturulmuş modern bir öğrenme/portföy arayüzünü içerir.

## Özellikler

- React 18 ve TypeScript tabanlı ön yüz
- Vite ile hızlı geliştirme deneyimi
- Tailwind CSS ve özel tema desteği
- Eğitim içerikleri, etkileşimli dersler ve kod örnekleri
- Figma tasarımına dayanan görsel düzen ve bileşen mimarisi

## Kurulum

1. Depoyu klonlayın veya kopyalayın.
2. Proje kök dizininde terminal açın.
3. Bağımlılıkları yüklemek için:

```bash
npm install
```

## Geliştirme Sunucusu

```bash
npm run dev
```

Tarayıcınızda açılan yerel URL üzerinden projeyi inceleyebilirsiniz.

## Proje Yapısı

- `src/main.tsx`: Uygulamanın giriş noktası
- `src/app/App.tsx`: Ana uygulama bileşeni
- `src/app/components/`: UI bileşenleri ve sayfa modülleri
- `src/styles/`: Stil ve tema dosyaları

## Build

```bash
npm run build
```

Üretim için optimize edilmiş çıktı `dist/` klasörüne yazılır.

## Notlar

- Bu proje `pnpm-workspace.yaml` içeren bir çalışma alanında bulunuyor, ancak `npm` komutlarıyla da çalışır.
- Figma tasarım kaynakları proje açıklamasında referans olarak yer alır.
